# 🎮 Voxel World - Megacity Edition

An interactive 3D web-based simulation game built using **Three.js** and **PointerLockControls**. This project combines the creativity of city-building mechanics with the freedom of a voxel-based sandbox explorer.

## 🚀 Key Features
- **🌆 Megacity Builder:** Build your dream metropolis! Place skyscrapers, houses, streets, traffic lights, and streetlights. Watch the city come alive with a dynamic day-night cycle and moving traffic (cars & motorcycles).
- **⛏️ Voxel Explorer:** Switch to a first-person perspective to explore a procedurally generated grid world, break blocks, and build new structures block-by-block.
- **🌧️ Weather System:** Toggle between different weather conditions in real-time, including Sunny, Cloudy, Rain, and Storms, complete with particle and fog effects.
- **✏️ Edit Mode:** Seamlessly select, drag to reposition, or delete existing buildings directly from the map.

## 🕹️ Controls
### City Builder Mode
- **Left Click (On Ground):** Build the selected object type.
- **Right Click / Middle Click + Drag:** Rotate/orbit the camera.
- **Mouse Wheel:** Zoom camera in and out.
- **Delete Key:** Remove the selected object (while Edit Mode is ON).

### Voxel Explorer Mode
- **Left Click Screen:** Lock mouse cursor (Pointer Lock).
- **W, A, S, D:** Move Forward, Left, Backward, and Right.
- **Spacebar:** Jump.
- **Left Click:** Break a block.
- **Right Click:** Place a block.
